# Rover H533 – Embedded Robotics Platform

## Overview
Personal robotics platform based on STM32H533RE.
Focus: structured firmware architecture, motor control, modular drivers.

## Architecture
- MCU: STM32H533RE
- IDE: STM32CubeIDE
- Language: C
- Firmware structure:
  - hardware/
  - firmware/
  - docs/

## Goals
- Clean driver abstraction
- Deterministic motor control
- Incremental AI integration (future phase)
